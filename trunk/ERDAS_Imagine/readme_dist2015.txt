Copy the x86/kea.dll to C:\Program Files\Hexagon\ERDAS IMAGINE 2015\usr\lib\Win32Release\rasterformats
and run C:\Program Files\Hexagon\ERDAS IMAGINE 2015\bin\Win32Release\configure_rf.exe

If you have 64 bit Imagine installed, also copy x64/kea.dll to C:\Program Files\Hexagon\ERDAS IMAGINE 2015\usr\lib\x64URelease\rasterformats
and run C:\Program Files\Hexagon\ERDAS IMAGINE 2015\bin\x64URelease\configure_rf.exe

and restart Imagine.
